## 脆弱なパスワード

``` sh
# - 7文字以下
# - 同じ文字だけで構成
# - 同じ種類の文字列だけで構成
(.{0,7}|(.)\2+|[a-z]+|[A-Z]+|[0-9]+)
```

``` sh
$ echo -e 'hoge\n@@@@@@@@@@@@\nbbbbbbbbb\n20040202' | grep -E '(.{0,7}|(.)\2+|[a-z]+|[A-Z]+|[0-9]+)'
hoge
@@@@@@@@@@@@
bbbbbbbbb
20040202
```

- 2種類以上とかを考えるのは複雑、プログラムコードで追加的なチェックが必要そう。。。

``` sh
# キャプチャして
$ echo hogeee | grep -Eo '(.)\1+'
eee
```

## 空行系

``` sh
$ echo '' | grep '^$'

# スペース・タブ・改行を無視する。
$ echo ' ' | grep '^\s$'

# 空行**以外**にマッチ。
$ echo 'ho\n \nge' | grep '\S'
ho
ge
```
